## Featured projects

### [prize-bond-checker](https://github.com/Irsalistic/prize-bond-checker)
A complete open-source CLI I built from scratch to solve a real everyday problem — checking Pakistani prize bonds against official draw results without manually searching thousands of numbers. It supports auto latest-draw detection, bulk CSV/Excel import, exact bond matching, and optional AI summaries through **Ollama, Gemini, or GPT** (works fully offline too). Packaged with **pytest**, clean project structure, and privacy-first local storage.

### [comfyui-dam-object-extractor](https://github.com/Irsalistic/comfyui-dam-object-extractor) · ⭐ Top repo
My most-starred project — a **ComfyUI custom node** for object extraction and masking using **DAM segmentation**. Built for generative pipelines where precise object isolation makes the difference between a rough output and a production-quality result.

### [pdf_chat](https://github.com/Irsalistic/pdf_chat)
A full-stack document AI setup: upload a PDF, ask questions, get streamed answers. Uses **FastAPI**, **Ollama**, **Redis** for conversation memory, and **Docker** for deployment. Private, self-hosted RAG — no vendor lock-in.

### [AI_lip_syncing](https://github.com/Irsalistic/AI_lip_syncing)
Production-oriented lip sync API — upload video, provide text, get back synchronized output with **multi-language TTS**. Video trimming, audio extraction, and model integration wrapped in a **FastAPI** service.

### [AI_sketch_to_art](https://github.com/Irsalistic/AI_sketch_to_art)
Turn rough sketches into polished art. **Stable Diffusion + ControlNet** with API-key auth, image preprocessing, and prompt-driven generation. One of my favorite demos of CV + GenAI working together.

### [AI_Resume_Optimizer](https://github.com/Irsalistic/AI_Resume_Optimizer)
LLM-powered resume analysis — upload **PDF or DOCX**, paste a job description, get tailored improvement suggestions via **Ollama, OpenAI, or Groq**. Built with **Streamlit** and **LangChain**.
